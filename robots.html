ROBODOG

Price Range$50 – $100

Description

Cujo is a multi-functional robot dog designed for kids, families, and hobbyists. It performs interactive actions, plays music, tells stories, and can be programmed for STEM learning.

Key Features

  • Dual control: remote + mobile app + some versions support voice commands
  • 20+ actions: walking, sitting, dancing, kung-fu, swimming motions, push-ups, handstands
  • STEM learning: supports Scratch, Arduino, Python (varies by model)
  • Sensors: LED expressive eyes, ultrasonic obstacle avoidance, IMU balancing
  • Entertainment: music, storytelling, singing
  • Optional water-bead launcher (some versions)

Additional Details

  • 8 high-speed micro-servos for smooth motion
  • Fast response time
  • Works on tile, wood, carpet, concrete
  • ESP32 controller with Wi-Fi + Bluetooth
  • Custom sound uploads (varies by version)
  • Battery: 7.4V or 12V lithium
  • Runtime: 45–90 min | Charge: 1.5–2 hr
  • Durable ABS shell, reinforced joints

SECURITY BOT

Price Range$50 – $75

Description

The R-69 is a mobile home-security robot with dual HD lenses, 360° viewing, and full app control. It functions as a moving security camera, pet monitor, and interactive home assistant.

Key Features

  • 360° dual-lens HD camera
  • Two-way audio
  • Remote control movement via mobile app
  • Motion detection + auto tracking
  • Night vision
  • Pet interaction mode
  • Compact size for tight spaces

Additional Details

  • Dual-lens system: wide-angle + PTZ
  • 2K resolution (varies by model)
  • Night vision range: 10–15 meters
  • Auto-tracking with adjustable sensitivity
  • Moves smoothly on tile, wood, low carpet
  • Runtime: 2–4 hours
  • App features: live view, patrol mode, snapshots, video recording
